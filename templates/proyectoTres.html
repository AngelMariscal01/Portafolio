Proyecto Grupal - “SoyHenry”:

Python | BigQuery | Storage | pandas | matplotlib | MySQL Workbench | SQL | Power BI | seaborn | Scikit-learn | XGBoost

Insightful Data Solutions:

Este proyecto es una plataforma de gestión de datos desarrollada como parte del programa de estudio en Henry, que permite la integración y análisis de grandes volúmenes de datos para facilitar la toma de decisiones en tiempo real. El proyecto fue realizado en equipo, y cada miembro tuvo un rol específico para asegurar la calidad y funcionalidad del sistema.

Mi Rol: Data Engineer - conexiones, infraestructura en la nube, deploy ML.

Fui responsable de la ingeniería de datos, lo que incluyó el diseño e implementación de conexiones entre múltiples servicios y el manejo de la infraestructura en la nube.

  • Google Cloud Run: Implementé contenedores para el despliegue de aplicaciones escalables.
  • Google Cloud Functions: Diseñé funciones serverless para procesar eventos y ejecutar lógica de negocio.
  • Cloud SQL & MySQL Workbench: Configuré y administré bases de datos para almacenar y gestionar los datos de manera eficiente.
  • Cloud Storage & BigQuery: Utilicé Google Cloud Storage para el almacenamiento de datos no estructurados y BigQuery para consultas analíticas de grandes volúmenes de datos.

    • Colaboración: Trabajo en equipo con herramientas como Slack y GitHub.
