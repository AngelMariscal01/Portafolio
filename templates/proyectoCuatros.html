Aplicación para el Acceso a Eventos

Python | Tkinter | JSON | GIT

La aplicación de gestión de acceso a eventos es una herramienta intuitiva y eficiente que permite a los organizadores de eventos registrar, modificar y eliminar invitados, así como gestionar su estado de asistencia. La interfaz gráfica es fácil de usar y proporciona funciones de búsqueda y filtrado para mejorar la experiencia del usuario. El sistema está diseñado para eventos donde es necesario llevar un control preciso de los asistentes, como conferencias, talleres, seminarios o reuniones privadas (Se recomienda usar algun tipo de lector de códigos como: QR, Barras, etc).

Características Principales:

  • Interfaz Gráfica de Usuario (GUI) Amigable: Utiliza Tkinter para crear una interfaz gráfica interactiva que es fácil de navegar y usar.
  • Gestión de Invitados: Permite ingresar nuevos invitados mediante un formulario que solicita el código y el nombre del invitado. Posibilidad de eliminar invitados seleccionados de la lista y del archivo JSON.
  • Registro de Asistencia: Funcionalidad para marcar la asistencia de un invitado ingresando su código directamente. Los usuarios pueden registrar la asistencia de los invitados seleccionados como "Asistido" o "Sin Asistir" con botones dedicados.
  • Búsqueda y Filtrado: Permite buscar invitados por código o nombre, mostrando resultados en tiempo real. Los invitados pueden ser filtrados para mostrar solo aquellos que han asistido o no.
  • Persistencia de Datos: Los datos de los invitados se almacenan en un archivo JSON, lo que permite guardar y cargar fácilmente los registros de la aplicación.
  • Feedback al Usuario: Mensajes de error y éxito proporcionan retroalimentación clara al usuario, mejorando la experiencia y usabilidad de la aplicación. Mensajes emergentes para confirmar acciones como la eliminación de invitados y la adición exitosa de nuevos registros.

Este proyecto es una solución efectiva para la gestión de eventos, ayudando a los organizadores a mantener un control organizado y preciso de los participantes. La aplicación es una excelente muestra de tus habilidades en desarrollo de aplicaciones GUI y manejo de datos en Python.
